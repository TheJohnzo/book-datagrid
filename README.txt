Simple AngularJS data grid display records of books.  

Features:

Filter by category, genre, or freetext.
Pagination.
Clicking a book will load a second page with additional details about that book.


TO INSTALL

*Clone repository into web server folder (e.g. htdocs)
*Whatever folder name you create will need to be updated in app.js so app can include templates from correct location. 
**app.js:L5  (default: /reedsy-challenge/)
*Angular and boostrap source is included here, for more maintainability consider package installer (composer or ruby gems)
